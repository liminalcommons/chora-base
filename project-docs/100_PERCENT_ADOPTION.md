## Adoption Approach: Awareness-First

### What is Awareness-First Adoption?

When existing infrastructure already implements SAP patterns (or when a capability is a declared dependency but not yet actively used), we use **awareness-first adoption**:

1. **Install Documentation:** Copy SAP artifacts to `docs/skilled-awareness/`
2. **Audit Compliance:** Check if existing code already follows patterns
3. **Update Discoverability:** Add capability entries to AGENTS.md
4. **Mark Status:** Tag as "awareness-only" if not actively used yet

### Benefits

- ‚úÖ **Zero Code Changes:** No risk of introducing defects
- ‚úÖ **Immediate Documentation:** Patterns available when needed
- ‚úÖ **Fast Adoption:** Can install multiple SAPs quickly
