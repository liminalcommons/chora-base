
## How to Adopt v1.1.0

### Option 1: Fresh Installation (Recommended for New Adopters)

**If you haven't adopted SAP-001 yet**:

1. **Install the protocol** (5 minutes):
   ```bash
   cd /path/to/your/repo
   python /path/to/chora-base/scripts/install-inbox-protocol.py \
     --repo github.com/liminalcommons/YOUR-REPO \
     --mode full \
     --capabilities "your-capability-1" "your-capability-2" \
     --contact-email your-email@example.com
   ```

2. **Configure capabilities** (2 minutes):
   - Edit `inbox/ecosystem/CAPABILITIES.yaml`
   - List what your repo can provide and receive

3. **Test the workflow** (3 minutes):
   ```bash
   # Check inbox
   python scripts/inbox-query.py --count-by-status

   # Generate test request
   python scripts/generate-coordination-request.py --interactive

   # Respond to test request
   python scripts/respond-to-coordination.py --help
   ```

4. **Register with ecosystem** (5 minutes):
   - Submit PR to chora-base with capability registry
   - Appears in ecosystem dashboard

**Total Time**: ~15 minutes (including configuration)

### Option 2: Upgrade Existing Installation

**If you already have SAP-001 v1.0.0**:

1. **Copy new tools** (2 minutes):
   ```bash
   # From chora-base repository
   cp scripts/inbox-query.py YOUR-REPO/scripts/
   cp scripts/respond-to-coordination.py YOUR-REPO/scripts/
   cp scripts/generate-coordination-request.py YOUR-REPO/scripts/
   cp -r scripts/inbox_generator/ YOUR-REPO/scripts/
   ```

2. **Update capability registry** (2 minutes):
   - Add SLA section to `inbox/ecosystem/CAPABILITIES.yaml`
   - Confirm response time commitments

3. **Update agent automation** (1 minute):
   - Add CLI commands to your `inbox/AGENTS.md`
   - Integrate into session startup routine

4. **Test upgraded workflow** (2 minutes):
   ```bash
   python scripts/inbox-query.py --count-by-status
   ```

**Total Time**: ~7 minutes

### Option 3: Observer Status (No Installation)

**If you want to monitor ecosystem coordination without active participation**:

- No installation required
- Monitor `chora-base/inbox/ecosystem/` for announcements
- Participate in quarterly reviews
- Upgrade to full participation when ready
